# Diskprotektorat
Better windows file explorer with actually usable features

**Installation **
1. Select directory when you want it to be installed
2. Clone the repo, while in the selected directory : git clone https://github.com/Chillosh/Diskprotektorat.git
3. Enter the repo : cd Diskprotektorat
4. Install it : python -m pip install .
5. Execute it : python -m diskprotektorat
